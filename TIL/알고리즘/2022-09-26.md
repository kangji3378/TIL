## 알고리즘 개발
ex> Search Engine Algorithm
    => PageRank Algorithm
    News Feed Algorithm
## 알고리즘 설계 및 분석
* 어떤 문제 p가 주어졌을 때
    * 컴퓨터로 문제 p를 해결할 수 있는가?
* 해결할 수 있다면 문제 p를 해결하는 알고리즘 A에 대하여
    * 알고리즘이 정확한가?
    * 알고리즘은 얼마나 좋은 알고리즘인가? (ex>리소스가 어느정도인가)
    * 알고리즘A보다 더 좋은 알고리즘은?
* 알고리즘이 좋다는 것은 어떻게 평가하는가?
    * 알고리즘이 수행되는 시간은?
    * 알고리즘이 사용하는 메모리의 양은?

## 알고리즘 설계

* 설계 기법(ex> 재귀,브루트포스,동적계획법...)
